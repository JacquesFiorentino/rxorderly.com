Android Beta

Thanks for testing RxOrderly! Follow the steps below to install the app on your Android device or tablet.

⬇ Download RxOrderly.apk

  1. Tap the download button above on your Android device.

    Open this page in Chrome or your device's default browser for the smoothest experience.

  2. Allow installation from this source.

    Because this is a beta (not from the Play Store), Android will ask for permission before installing. The exact prompt varies by manufacturer and Android version, but you'll see one of these:

    Chrome prompt: Tap Settings when Chrome asks, then enable Allow from this source, then go back and tap Install.

    Files app prompt: Tap Install anyway or Allow when your file manager asks about installing unknown apps.

    Settings path (if prompted manually): Go to Settings → Apps → Special app access → Install unknown apps, find your browser or file manager, and enable it.

  3. Tap Install when the system installer appears.

  4. Open RxOrderly from your home screen or app drawer and set up your medications.
